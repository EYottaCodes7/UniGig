## 🚀 About Unbroke

Unbroke is a premier student-to-student marketplace built for the **AAU GDG Hackathon 2026**. Designed specifically for campus life, Unbroke empowers students to turn their skills into income. From tech repairs and dorm deliveries to academic tutoring and second-hand sales, Unbroke connects students who need help with students who can provide it.

## ✨ Features

- **Student-First Marketplace**: Tailored specifically to the needs of university students.
- **Skill Exchange & Gig Work**: Post services, offer your skills, and earn money hassle-free.
- **Campus Convenience**: Custom delivery options like "Dorm Delivery" to make on-campus trading a breeze.
- **Trusted Community**: Engage securely with verified peers.
- **Modern & Fast UI**: Built with Next.js and animated with Framer Motion for a seamless, application-like experience.

## 🛠 Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Language**: TypeScript
- **Backend & Database**: [Supabase](https://supabase.com/)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **UI Components**: shadcn/ui

## 📦 Getting Started

1. **Clone the repository:**

   ```bash
   git clone https://github.com/dayniia/aau-hackathon.git
   cd aau-hackathon/UniGig
   ```

2. **Install dependencies:**

   ```bash
   npm install
   # or
   pnpm install
   ```

3. **Run the development server:**

   ```bash
   npm run dev
   # or
   pnpm dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
